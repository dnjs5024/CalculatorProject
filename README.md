기능
-
---
>사칙연산
-   기본적인 + , - , / , * 입력창에 직접 입력가능, 버튼 클릭 해서도 입력 가능
- 소수 계산 기능
- 한 번에 여러 사칙연산 가능 ex) 5+4*2-2
>초기화
-  " C " 버튼 클릭 시 입력 데이터 전체 삭제
-  " 지우기 "" 버튼 클릭 시 입력데이터 한 개 삭제
>계산된 수 화면 보이기
- 계산 후 데이터 화면에 넣어줌
>숫자와 사칙연산 기호만 입력받기
-  사칙연산 기호 2개가 연속으로 입력 시 나중에 들어온 기호로 변경 ex) -+ --> + 로 변경 
-  ctrl + v 로 입력했을 때 문자열 전체 검사
-  마지막에 기호 입력 시 제거
